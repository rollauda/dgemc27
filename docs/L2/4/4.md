# Le jeu de l’île

import useBaseUrl from '@docusaurus/useBaseUrl';

## écriture d'une constitution 

:::note[Scénario]
Une pandémie a décimé 99,99 % de la population mondiale et vous êtes sur un bateau où tout le monde est immunisé contre le virus. Après un long voyage, vous échouez sur une île déserte qui contient assez de ressources pour survivre. Vous savez que vous et les générations suivantes resterez sur cette île. Vous devrez donc fonder une nouvelle société à partir de rien : imaginer le type de régime politique que qous adopterez, mais aussi décider d'un système juridique. 
**Évaluation finale : écriture de la constitution de votre île**.
:::
